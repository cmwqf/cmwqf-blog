# Description

要带n个物品，要求：

汉堡：偶数个；可乐：0/1个；鸡腿：0/1/2个；蜜桃：奇数个；

鸡块：4的倍数个；包子0/1/2/3个；土豆：0/1个；面包：3的倍数个。

求所有情况的方案数，对10007取模。

**注意：**$1<=n<=10^{500}$

<!--more-->

# Solution

考虑生成函数。

我们对每一种情况搞一个生成函数，其中$x^i$指选i个，前面的系数就是选$i$个的方案数。最后，将它们乘起来，取第$n$项的系数就好了。
